# Phase 6: Reporting and Export

## Goal
Provide reward reporting, member export, and filtered export functionality.

## Reward Report
- Filter by date range.
- Filter by member.
- Filter by promotion.
- Display achieved rewards clearly with earned date and tier details.

## Member Export
- Export full member list.
- Export filtered member results.

## Reward Export
- Export filtered reward reports.
- Support CSV and XLSX output.

## Deliverables
- Report controller and query layer.
- Export classes using the same filter rules as the UI.

# Phase 1: Foundation

## Goal
Set up the Laravel 11 project structure, shared layout, auth protection, and required infrastructure for the membership system.

## Implementation
- Create a fresh Laravel 11 application.
- Configure database, filesystem storage, and environment variables.
- Install and configure authentication for admin-protected actions.
- Add a shared admin layout with reusable form and table components.
- Choose a UI baseline suitable for the test, such as Bootstrap.
- Prepare export support for CSV/XLSX reports.

## Deliverables
- Working Laravel 11 app skeleton.
- Protected admin route group.
- Shared UI shell for all member and reward pages.
